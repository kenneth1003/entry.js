You can config Entry, as following 

        var foo = new Entry({
          theshold: 420,  // in pixel, showing theshold from bottom, default is 200
          duration: 2,  // in second, transition time, default is .5
        });
        foo.init();

You can also set delay time of showing in the html attribute like: 

          <div class="entryjs" data-entryjs-delay="1000" // in millisecond ...

no data-entryjs-delay
