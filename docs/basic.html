After adding the entry.js and entry.css in you project,
add "entryjs" in the classlist of the element,
and the following in the Javascript, 

        var foo = new Entry();
        foo.init();  // execute after document on ready
then you are good to go :)

If you want element enters in different ways, (defalut is fade-in)
add an extra "entryjs-{property}" class.

ex: "entryjs-right", "entryjs-top", "entryjs-no-opacity"
