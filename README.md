# entry.js
light weight frontend plugin

## Getting Started

```
// your .html

<link rel="stylesheet" href="/css/entry.css?k0912">
<script src="/javascript/entry.min.js"></script>

// element
<div class="entryjs">

// you .js
 var foo = new Entry();
 foo.init();  // execute after document on ready
 
```
